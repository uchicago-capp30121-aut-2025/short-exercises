Short Exercises #1
------------------

These are the files for the Short Exercises #1.

- se1.py: You will write your solutions in this file.

- test_se1.py: The automated tests. Run all of the automated tests using:

  $ uv run pytest -v -x test_se1.py

  See the "Testing Your Code" page on the course website for
  more detailed instructions. 

- pytest.ini: A configuration file that you can safely ignore.

- README.txt: This file.

- helpers.py: A file containing helper functions for the test. 
  DO NOT MODIFY this file.
